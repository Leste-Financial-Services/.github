# ☁️ Cloud Patterns (Azure Ecosystem)

Define como aproveitar ao máximo o ecossistema Azure.

## 1. Prioridade Cloud
- O desenvolvimento deve priorizar serviços PaaS/SaaS do **Microsoft Azure**.
- Azure DevOps para repositórios e backlog.

## 2. Cost-Efficiency
- Priorizar soluções que reduzam consumo de nuvem sem comprometer SLA.
- Utilizar **Azure Functions** ou **App Service (B1/S1+)** para cargas variáveis.

## 3. Serverless vs Realtime
- Escolha **Azure Functions (Timer Trigger)** para jobs agendados.
- Use **Azure Service Bus** para comunicação assíncrona desacoplada entre sistemas.
