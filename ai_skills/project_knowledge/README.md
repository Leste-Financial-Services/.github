# Leste Project Knowledge

This folder stores the cross-project knowledge index generated from repositories under `D:\Source\Devops`.

Main file:

- `leste_projects_index.md` — generated index with project summaries, stack classification, frontend production endpoints, backend endpoint hints, and reuse candidates.

Regenerate with:

```powershell
.\Update-LesteProjectKnowledge.ps1 -RootPath "D:\Source\Devops"
```
