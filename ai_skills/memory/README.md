# 🧠 Memória Persistente do Copilot

## Propósito
Este diretório contém o sistema de memória persistente do Copilot CLI.
Ele garante que o histórico de atividades e decisões sobreviva a reinicializações
do laptop, trocas de sessão e qualquer interrupção.

## ⚠️ REGRA OBRIGATÓRIA PARA TODA SESSÃO
1. **Ao iniciar qualquer sessão**, leia `activity_log.md` para se contextualizar.
2. **Ao concluir qualquer atividade**, registre no `activity_log.md`.
3. **Nunca apague** entradas anteriores — apenas adicione novas.

## Arquivos
| Arquivo | Descrição |
|---------|-----------|
| `activity_log.md` | Log cronológico de todas as atividades realizadas |
| `README.md` | Este arquivo — regras do sistema de memória |
