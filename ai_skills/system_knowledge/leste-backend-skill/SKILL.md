This skill provides specialized knowledge for Leste's backend services.

- **Patterns**: Clean Architecture (API, Domain, Infrastructure), `ConfigureDomain`, `ConfigureInfra`.
- **Packages**: `Leste.Logging`, `Leste.Utils`.
